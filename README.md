<p>Converteer eenvoudig xml bestanden naar json binnen enkele seconden</p>

<h1>Module 2 To Do list api handleiding</h1>
<h2>To do api Crud item</h2>
<p>Deze api is gebouwd voor tip top bv. Dit bedrijf wenste voor een agile achtige item crud systeen</p>
<h3>Endpoints:</h3>
  <li>	GET		/items					Verkrijg alle items></li>
  <li>GET		/items/{id}				Verkrijg item met id</li>
  <li>POST		/items					Item aanmaken</li>
  <li>PUT		/items/{id}				Item updaten</li>
  <li>DELETE	/items/{id}				Item verwijderen</li>
